How to use Tab Group and Tab Group Shortcut

Use Ctrl key or Shift key to select multiple tabs.

Press Ctrl G to group selected tabs. (extension feature)

You can also Right Click group icon to change name and color.

Left Click will toggle minimization of the group.

You need to refresh the existing tabs to use shortcut when first installed.

Drag the tab(s) to add to / remove from group.

You can ungroup with Ctrl G too.

Because of security reasons of the browser, Ctrl G is not available when focusing at some tabs.

e.g. Chrome Internal chrome://, New Tab chrome://newtab, and Chrome Web Store pages.

You can use Ctrl Q command to Force Group / Ungroup.

You can turn off Ctrl Q at Options page. (Right Click extension icon -> Options)
